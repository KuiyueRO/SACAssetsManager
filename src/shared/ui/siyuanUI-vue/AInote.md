# 这个区段由开发者编写,未经允许禁止AI修改

思源UI Vue组件库需要遵循思源笔记的设计语言和风格指南，确保在不同主题下UI保持一致性。

## 设计原则
1. 使用思源的CSS变量系统，避免硬编码颜色、尺寸等
2. 组件应在不同的主题（包括明亮和暗黑主题）下自动适应
3. 遵循思源笔记的交互设计模式
4. 保持简洁，专注于功能实现
